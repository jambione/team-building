# /adr-create

Create a new Architecture Decision Record. data leads. Usage: /adr-create <describe the decision context>

Load personas from `.clinerules/TNG/`. Read `architecture-decision-records.md` and `architecture-principles.md` before starting.

**data** leads: draft a complete ADR (Title, Status, Context, Decision, Rationale, Alternatives Considered, Consequences, Review Date). Append to `architecture-decision-records.md`. End with: `data returns control to picard. [arch-design-complete]`

**worf** supports: security implications.
**crusher** supports: reliability and operational consequences.

picard delivers: confirmation ADR is consistent with KB, index updated. Close with **"Make it so!"**
